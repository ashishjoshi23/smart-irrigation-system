# Components List

## Main Components
1. **Raspberry Pi 3 Model B** - Main controller
2. **LCD Display (16x2)** - LM016L for status display
3. **LM35 Temperature Sensor** - Analog temperature sensing
4. **Soil Moisture Sensor** - Measures soil water content
5. **Rain Sensor** - Detects rainfall
6. **DC Motor (12V)** - Water pump motor
7. **Relay Module** - Controls motor ON/OFF
8. **ADC Module (MCP3008)** - Analog to Digital Converter
9. **GSM Module (SIM800L)** - Sends SMS alerts
10. **Power Supply** - 5V 2A adapter

## Additional Components
- Breadboard and jumper wires
- Resistors (for voltage divider if needed)
- Capacitors (for power filtering)
- Potentiometer (for LCD contrast)
- Casing/Enclosure

## Software Requirements
- Raspberry Pi OS
- Python 3.x
- RPi.GPIO library
- spidev library
- Proteus 8.10 (for simulation)
